# PDF Viewer
Aplicação cujo propósito é visualizar arquivos de um dispostivo móvel.

## Funcionalidades
- [x] Ver arquivos recentes
- [x] Ver todos arquivos do dispositivo
- [x] Pesquisar arquivos do dispositivo
- [x] Favoritar arquivo
- [x] Compartilhar arquivo
- [ ] Salvar cópia
- [x] Imprimir
